# WASM Error Codes V1

**Status**: Draft

## Categories

- ABI: `WASM_ABI_*`
- Verify: `WASM_VERIFY_*`
- Capability: `WASM_CAPABILITY_*`
- Determinism: `WASM_DETERMINISM_*`
- Resource: `WASM_RESOURCE_*`
- Receipt Binding: `WASM_RECEIPT_BINDING_*`

## Stability Rule

Error code strings are contract surface and must be backward-compatible.
