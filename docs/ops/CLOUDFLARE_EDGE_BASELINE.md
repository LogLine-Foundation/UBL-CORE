# Cloudflare Edge Baseline

**Status**: active  
**Owner**: Ops + Security  
**Last reviewed**: 2026-02-21

## Required Rules

- Rate limit: `/v1/receipts`
- Rate limit: `/v1/chips`

## Rule Registry

Store active rule IDs/names in one of:

- env `UBL_CLOUDFLARE_RATE_LIMIT_RULES`
- `${UBL_BASE_DIR}/state/cloudflare_rate_limit.json` generated by bootstrap

## TLS Trust Model

Current trust model relies on TLS termination at Cloudflare edge.
If direct/off-Cloudflare ingress is introduced, dedicated PKI/certificate governance is required before go-live.
